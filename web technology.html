LAB 01-INTRODUCTION TO GITHUB


gitHub uses Git to enable collaborative software

development by allowing users to store, track, and merge code changes in online repositories.

It function s a cloud-based platform for creating,sharing,

and managing code and other project files, facilitating teamwork by enabling parallel development on

separate "branches" before merging into a main project.

BENEFITS OF GITHUB

benefits:
  • Version Control and Collaboration
  • Centralized Code Hosting
  • Project Management Tools
  • Community and Open Source
  • Showcasing Work and Networking
The gitwas founded in 2008 by Tom Preston-Werner, Chris Wanstrath, PJ Hyett, and Scott Chacon.

It started as a platform for hosting Git repositories and quickly grew into one of the most popular tools for version control
and collaborative software development.
